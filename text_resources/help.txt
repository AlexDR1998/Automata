========================================================
This program generates and displays 2D cellular automata
See user guide for more detailed explanations 

Enter	-- Generate and run new random rule
r 	-- Run current rule from new initial condition
c 	-- Run current rule from end of last simulation
s 	-- Save current rule
l	-- Load rule and generate animation
g 	-- Change rule generation method
y 	-- Change parameters for rule generation

+,-,*	-- Element-wise arithmetic on 2 rules
a 	-- Element-wise average of 2 rules
z	-- Zip 2 rules together
b 	-- Splice 2 rules together

p	-- Perturb current rule and generate animation
m	-- Choose from 4 mutations of current rule
x	-- Shuffle current rule
v	-- Roll current rule
w 	-- Smooth current rule
i 	-- Invert current rule

sm	-- Gaussian blur of simulation
t 	-- Transpose output
fft	-- Display FFT of simulation
2d 	-- Display sum along given axis
stat	-- Display stationary slices

info	-- Displays how many possible rules there are
q	-- Quit
h	-- Display this help message
